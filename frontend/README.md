# Canal Tech - Frontend

Interface do usuário para o portal de notícias Canal Tech.

## 🚀 Como rodar

### Desenvolvimento Local
```bash
# Instalar dependências
npm install

# Rodar servidor
npm run dev
```

### Com Docker
```bash
docker compose up frontend
```

Aplicação: `http://localhost:3000`

## 🧪 Testes

```bash
npm test
```

## 🛠️ Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- Jest + Testing Library
